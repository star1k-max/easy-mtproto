# 📡 MTProxy Installer for Telegram

Автоматическая установка личного MTProxy-сервера для Telegram с генерацией ссылки и QR-кода.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 📖 Описание

Этот скрипт устанавливает [MTProxy](https://github.com/TelegramMessenger/MTProxy) — официальный прокси-сервер для Telegram. 
Всё, что нужно для быстрого запуска собственного прокси: установка зависимостей, сборка из исходников, настройка systemd, открытие порта в firewall и создание удобной ссылки для подключения. 
В конце работы скрипт выводит:
- готовую ссылку `t.me/proxy` для Telegram;
- QR-код прямо в терминале;
- PNG‑файл с QR-кодом и текстовый файл со ссылкой.

Скрипт полностью повторяет рекомендации из [официального README](https://github.com/TelegramMessenger/MTProxy) и обновляет конфигурацию Telegram ежедневно через cron.

## ✨ Особенности

- 🔧 **Сборка из исходников** — всегда актуальная версия.
- ⚙️ **Автозапуск через systemd** — прокси стартует вместе с сервером.
- 🔥 **Firewall** — автоматически открывает выбранный порт в UFW.
- 🔗 **Ссылка-приглашение** — мгновенная настройка в Telegram.
- 📱 **QR‑код** — удобно сканировать с телефона.
- 🧹 **Чистота** — создаёт всего один сервис и несколько файлов, без мусора.

## 📋 Требования

- VPS (или любой сервер) с **Ubuntu 20.04/22.04/24.04** или **Debian 11/12**.
- Публичный IPv4-адрес.
- Доступ **root** или возможность выполнять команды с `sudo`.
- Свободный TCP-порт (обычно `443`, но можно любой).

> **Важно**: скрипт не проверяет, занят ли выбранный порт. Если на сервере уже работает nginx, Apache или другое ПО на порту 443, укажите другой порт (например, `8443`).

## 🚀 Быстрая установка

Выполните команду **от root**:

```bash
curl -sSL https://raw.githubusercontent.com/star1k-max/easy-mtproto/main/install-mtproxy.sh | bash
```
## ❌ Удаление следов скрипта

```bash
curl -sSL https://raw.githubusercontent.com/star1k-max/easy-mtproto/main/uninstall-mtproxy.sh | bash
```
